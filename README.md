Interactive Python Calculator with loop and random massage.

This is simple console-based calculator written in Python.



* Addition 
* Subtraction 
* Multiplication 
* Division 
* Finding the remainder in division 
* Division without remainder 
* Raising a number to a certain power
* Loop mode (continue calculation without restarting)
* random massage after each calculation 



How to run

1. Make sure you have Python installed
2. Run the file
3. Get an answer
4. Use again
